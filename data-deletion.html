1. 데이터 삭제 요청 절차

HyperClick은 사용자가 앱 사용 중 수집된 데이터의 삭제를 요청할 수 있는 권리를 보장합니다. 아래 단계를 따라 요청해 주세요.

  1. 이메일 작성: 아래 이메일 주소로 데이터 삭제 요청 메일을 보내주세요.
    hyperclick.customer@gmail.com
  2. 제목: HyperClick 데이터 삭제 요청
  3. 본문에 포함할 정보:
    • Google Play 광고 ID (AAID) — 선택 사항. Android 설정 → Google → 광고 → 광고 ID에서 확인 가능
    • 대략적인 앱 사용 기간 또는 마지막 사용 시점
    • 삭제를 원하는 데이터 유형 (전체 또는 일부)
  4. 처리 기간: 요청 접수 후 영업일 기준 30일 이내에 처리 완료 메일을 발송해 드립니다.

앱 내에서 직접 요청하기

앱을 열어 설정(⚙️) → 문의하기를 선택하면 위 이메일 주소가 자동으로 입력된 메일 작성 화면이 열립니다.

2. 삭제 및 보관되는 데이터 유형

HyperClick은 회원가입 없이 익명으로 작동하며, 사용자 식별 가능한 개인정보(이름, 연락처, 주소 등)를 수집하지 않습니다. 다음은 광고 및 분석 SDK가 수집하는 데이터입니다.

데이터 유형 수집 주체 삭제 여부 추가 보관 기간
광고 ID (AAID) Google AdMob 요청 시 즉시 삭제 최대 60일 (Google 정책에 따라 익명 집계 데이터로 보관)
App Instance ID, 앱 상호작용 이벤트 Firebase Analytics 요청 시 30일 이내 삭제 최대 60일 (백업 시스템 자동 삭제 주기)
크래시 로그, 진단 데이터 Firebase Crashlytics 요청 시 즉시 삭제 최대 90일 (Crashlytics 자동 보관 정책)
대략적 위치 (IP 기반) Google AdMob 영구 저장하지 않음 광고 응답 즉시 폐기
매크로/설정 데이터 (로컬 저장) 사용자 기기 앱 삭제 시 자동 삭제 없음 (서버 전송 안 함)

즉시 삭제 가능한 셀프 조치

  • 광고 ID 재설정: Android 설정 → Google → 광고 → 광고 ID 재설정 또는 광고 ID 삭제
  • 앱 데이터 삭제: Android 설정 → 앱 → HyperClick → 저장공간 → 데이터 삭제
  • 앱 제거: 앱을 제거하면 Firebase Instance ID는 즉시 무효화됩니다.

HyperClick Data Deletion Request

Last Updated: 2026-05-12

1. How to Request Data Deletion

HyperClick respects your right to request deletion of any data collected through your use of the app. Follow the steps below.

  1. Send an email to the address below.
    hyperclick.customer@gmail.com
  2. Subject: HyperClick Data Deletion Request
  3. Include in the body:
    • Your Google Advertising ID (AAID) — optional. Found under Android Settings → Google → Ads → Advertising ID.
    • Approximate usage period or last-used date.
    • Which data types you want deleted (all or specific).
  4. Response time: We will confirm completion within 30 business days of receiving your request.

Request from inside the app

Open the app, go to Settings (⚙️) → Contact Us, and a pre-filled email draft will open.

2. Data Types Deleted and Retention Period

HyperClick operates anonymously without account registration and does not collect personally identifiable information (name, phone number, address, etc.). The following is collected by ad and analytics SDKs.

Data Type Collected By Deletion Retention Period
Advertising ID (AAID) Google AdMob Deleted immediately on request Up to 60 days (anonymized aggregate data per Google policy)
App Instance ID, app interaction events Firebase Analytics Deleted within 30 days of request Up to 60 days (backup system rotation)
Crash logs, diagnostic data Firebase Crashlytics Deleted immediately on request Up to 90 days (Crashlytics default retention)
Approximate location (IP-based) Google AdMob Not stored permanently Discarded immediately after ad response
Macro / config data (local only) User's device Removed when app is uninstalled None (never transmitted to server)

Self-service actions

  • Reset Advertising ID: Android Settings → Google → Ads → Reset advertising ID or Delete advertising ID
  • Clear app data: Android Settings → Apps → HyperClick → Storage → Clear data
  • Uninstall the app: Uninstalling immediately invalidates the Firebase Instance ID.
