HyperClick 개인정보처리방침

앱 이름: HyperClick - Auto App Clicker
개발자: Geosys co. ltd.
연락처: hyperclick.customer@gmail.com
시행일: 2026-05-13

Geosys co. ltd.(이하 "회사")는 HyperClick(이하 "앱")을 운영하면서 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 및 Google Play 데이터 보안 정책을 준수합니다. 본 방침은 앱이 수집하는 정보, 수집 방법, 이용 목적, 보관 기간 및 이용자의 권리에 관한 사항을 안내합니다.

1. 수집하는 정보

앱은 회원가입 없이 익명으로 동작하며, 이름·이메일·전화번호·주소 등 직접적인 식별 정보를 수집하지 않습니다. 다만 광고 및 안정성 유지를 위해 다음 데이터가 자동 수집됩니다.

데이터 유형 수집 주체 수집 목적 보관 기간
광고 ID (AAID) Google AdMob 맞춤 광고 표시 및 광고 실적 측정 최대 60일 (Google 정책 기준)
대략적 위치 (IP 기반) Google AdMob / Firebase Analytics 지역별 광고 타겟팅 및 사용 통계 일시적 처리 (광고 응답 후 즉시 폐기)
App Instance ID, 앱 상호작용 이벤트 Firebase Analytics 기능 사용 분석, 앱 개선 최대 14개월
크래시 로그, 진단 데이터 Firebase Crashlytics 오류 진단 및 안정성 개선 최대 90일
매크로 / 설정 데이터 사용자 기기 (로컬 저장) 앱 기능 제공 앱 삭제 시 자동 폐기

2. 제3자 제공 및 공유

회사는 이용자의 개인정보를 제3자에게 별도로 판매하지 않습니다. 다만 아래의 광고 및 분석 SDK는 데이터를 자동으로 처리·전송합니다.

  • Google AdMob: 광고 ID는 광고 입찰(RTB) 과정에서 광고 네트워크 및 광고주에게 전달될 수 있습니다.
    관련 정책: https://policies.google.com/technologies/ads
  • Firebase (Google): Analytics 및 Crashlytics 데이터는 Google에 의해 처리됩니다. Google은 본 서비스의 데이터 처리자로 동작합니다.
    관련 정책: https://firebase.google.com/support/privacy

3. 권한 및 사용 목적

권한사용 목적
INTERNET광고 표시, 크래시 보고, YouTube 영상 정보 조회
SYSTEM_ALERT_WINDOW플로팅 위젯(오버레이) 표시

ACCESSIBILITY_SERVICE사용자가 설정한 좌표에 자동 터치/제스처 수행. 비밀번호·메시지 등 민감 정보를 수집·전송하지 않습니다.

4. 이용자의 권리

  • 광고 ID 재설정·삭제: Android 설정 → Google → 광고에서 직접 재설정/삭제 가능
  • 앱 데이터 초기화: Android 설정 → 앱 → HyperClick → 저장공간 → 데이터 삭제
  • 앱 제거: 앱 제거 시 Firebase Instance ID는 즉시 무효화됩니다.
  • 서버 데이터 삭제 요청: 아래 데이터 삭제 페이지에서 절차를 확인해 주세요.
    데이터 삭제 요청 페이지 →

5. 아동 보호 (COPPA)

앱은 만 13세 미만 아동을 직접적으로 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다. 보호자가 아동의 개인정보가 수집되었다고 판단되는 경우 즉시 아래 이메일로 연락해 주시기 바랍니다.

6. GDPR / CCPA

유럽 경제 지역(EEA), 영국, 캘리포니아 거주 이용자는 GDPR / CCPA에 따라 정보 열람, 정정, 삭제, 처리 제한 요청 권리를 가집니다. 행사를 원하시는 경우 아래 이메일로 요청해 주십시오. 회사는 요청 접수 후 30일 이내에 처리 결과를 회신합니다.

7. 보안

앱의 모든 외부 통신은 HTTPS(TLS)로 암호화 전송됩니다. 회사는 합리적인 관리적·기술적 조치를 통해 이용자 정보를 보호합니다.

8. 정책 변경

본 방침이 변경될 경우 변경 사항은 본 페이지에 게시되며, 중요한 변경은 앱 내 공지 또는 새 버전 출시 시 안내됩니다.
